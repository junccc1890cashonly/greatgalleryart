# GalleryArt Design Template Recommendations

## 目标设计方向

本产品建议整体视觉关键词为：

- 简洁
- 高级
- 安静
- 留白
- 编辑感
- 收藏馆气质

不建议做成：

- 传统素材管理后台
- 重运营感瀑布流
- 过于科技感的 AI 工具面板

## 推荐模板数量

当前推荐 `4` 套方向模板，均适合“简洁、高级、有留白”的目标，但气质不同，可作为首页和核心页面的设计母版。

## 模板 1：Editorial Minimal

### 适用场景

- 首页
- Gallery 首页
- Prompt Studio

### 风格特征

- 大面积留白
- 细字重排版
- 卡片边框极轻
- 以内容本身作为视觉主体

### 视觉建议

- 主色：暖白、炭黑、雾灰
- 点缀色：低饱和鼠尾草绿或浅金
- 字体气质：高端杂志、画册、展览导视

### 为什么推荐

这个方向最适合“审美收藏 + AI 创作辅助”的定位，不像工具，更像一本可交互的灵感档案。

## 模板 2：Quiet Luxury Grid

### 适用场景

- Collection 页面
- 图片详情页
- 历史 Prompt 页面

### 风格特征

- 强网格
- 充足边距
- 精细分隔线
- 层级克制

### 视觉建议

- 主色：米白、沙灰、深棕、墨黑
- 大图与文字比例约 `7:3`
- 图像裁切统一，避免页面太碎

### 为什么推荐

它有“品牌手册”和“高端目录册”的感觉，适合承载大量图片但不显拥挤。

## 模板 3：Museum Archive

### 适用场景

- 图片详情
- Collection 详情
- 标签浏览页

### 风格特征

- 类策展式布局
- 图文说明关系清晰
- 信息密度低
- 每一张图像都像展品

### 视觉建议

- 主色：骨白、石灰、深灰蓝
- 使用编号、注释、小标题提升“档案感”
- 鼠标 hover 动画尽量轻

### 为什么推荐

适合把普通图片收藏做出“策展”气质，非常贴合你这个产品的审美方向。

## 模板 4：Soft Modern Mono

### 适用场景

- 空状态页面
- Prompt 生成结果页
- onboarding

### 风格特征

- 单色系或近单色系
- 轻微柔化圆角
- 组件数量少
- 更现代、更数字化

### 视觉建议

- 主色：白、浅灰、黑
- 点缀色只保留 1 个，例如深橄榄绿
- 用局部阴影代替复杂背景

### 为什么推荐

如果你希望整体更像现代创意 SaaS，而不是展览画册，这套会更平衡。

## 综合建议

如果只选一个主方向，建议优先：

`模板 1 Editorial Minimal`

原因：

- 最能体现简洁和高级感
- 最容易做出留白与呼吸感
- 与图片内容天然兼容
- 与 AI prompt 场景结合时不突兀

如果想要一个“主模板 + 辅模板”组合，建议：

- 主模板：`Editorial Minimal`
- 辅模板：`Quiet Luxury Grid`

组合方式建议：

- 首页 / Prompt Studio 用 `Editorial Minimal`
- Gallery / Collection 用 `Quiet Luxury Grid`

## 页面层面的具体建议

### 1. Gallery 首页

- 不建议一上来就做很密的瀑布流
- 建议用规整网格 + 适度留白
- 顶部筛选区域保持单行、低对比

### 2. Collection 详情页

- 封面图占大面积
- 文字信息靠左或靠下，避免堆太多控制项
- 操作按钮弱化，只保留关键动作

### 3. Prompt Studio

- 左侧参考图，右侧输出区
- 中间留出明显呼吸空间
- 输出文本尽量排版成“编辑稿”而不是代码面板

### 4. 图片详情

- 以图片为核心，元信息作为次级层
- 标签和 `hashtag` 用轻样式，不抢主体

## 不推荐的模板方向

- Dribbble 风高饱和渐变卡片风
- 典型 AI 工具紫色发光风
- 重控制台感的深色后台风
- 过于拟物和装饰性的艺术网站风

## 下一步建议

如果进入设计或原型阶段，建议先从以下 `2` 套模板做首页和 Gallery：

1. `Editorial Minimal`
2. `Quiet Luxury Grid`

这样可以快速判断你更偏“策展画册感”还是“高端产品感”。
